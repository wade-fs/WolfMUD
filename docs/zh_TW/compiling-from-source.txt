
名稱

  從源代碼編譯 WolfMUD

摘要

  本文檔描述了從源代碼編譯 WolfMUD 的詳細信息。此信息
  可以來自源代碼下載之一，也可以來自 Git 克隆。

說明

  假設已經下載並安裝了 Go。Go 環境
  可以從 https://golang.org 獲得，也可以從您的
  套件管理器。

  編譯 WolfMUD 至少需要 Go 1.18(Wade: 目前使用版本 1.19.2)。建議使用最新的
  Go 版本。

在 LINUX 或 WINDOWS 上編譯

  要編譯 WolfMUD，請打開終端或命令窗口並更改為
  未歸檔源碼的目錄，或 Git 克隆後的目錄，即 WolfMUD/ 。
  日後默認 WolfMUD/ 即為源碼根目錄。
  在命令提示符下執行命令：

    go build -o bin ./...

  編譯完的 Mud 服務器可執行文件將放在 bin 目錄中。有關詳細信息
  運行服務器請參閱 running-the-server.txt 了解詳細信息。
